ANIMATE YOUR CREATIONS

A game changer in the voxel world. With our own Skeleton System, VoxEditor allows you to rig and animate anything that you want using the timeline with a easy-to-use keyframing mode.

Features

  • Animation! Bring your creation to life
  • Skeleton System
  • Cut & Rig
  • Powerful Timeline
  • Keyframing interpolation
  • Curves
  • Animations integrated in one file

Skeleton system

The Skeleton lets you rig your model using a tree system with dependencies mixing the 3D model with the bone ready for animate it in the Timeline.

Timeline

VoxEditor is the first voxel software that allows you not only to rig your model with the Skeleton system but also to create animations.

Create all the animations that you want to make a better experience for your model inside the game.

Animate all your models with our Timeline using keyframing interpolation in a friendly interface.

Move your keyframes, stretch the interpolation and add curves for a more polished animation.

Library

Import all types of files into the Library to assign them to a bone in the Skeleton system.

You can import .obj, .vox and .vxm.
