Modeling overview

POWERFUL & EASY TO USE MODELING TOOLS

Make the most of your creativity! From begginers to experts, anyone can make amazing creations thanks to our robust set of tools and our user-friendly interface.

Features

  • All-in-One software (Model, Rig and Animate)
  • Incredible performance
  • Available for Windows and Mac
  • Amazing voxel creation tools
  • High level of detail models
  • Advanced Renderer
  • Versatile Color Palette
  • Friendly 3D enviroment

Tool Bar

  • Select different areas of your model
  • Create individuals voxels
  • Create amazing models from scratch
  • Fill the box with voxels to start crafting
  • Put color to your model
  • Fill the walls with voxels
  • Erase voxel in different ways
  • Paint directly to the voxel
  • Add glow to your voxel with the Emisive Wand

Menu Bar

  • Toggle within the Modeling and the Skeleton screen
  • Use Mirror in all the axis for symmetrical models
  • Make rotation of your work in all the axis
  • View your work in Voxels, Quads or with our powerful Render
  • Increase or decrease the amount of voxels for more or less details
  • Crop your model so you don’t waste unused space
  • Set up the Canvas. It can be small or super big
  • Reset the camera

Color Palette

  • Choose easily any color for your work
  • Save your own palette or import an existing one

Import / Export

VoxEditor import and export in all the standard formats:

  • .obj
  • .vox
  • fbx
  • .vxm (VoxEditor format)
